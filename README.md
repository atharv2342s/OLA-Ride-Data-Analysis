# OLA Ride Data Analysis Dashboard

## Project Overview

This project focuses on analyzing **OLA ride booking data** to understand booking trends, revenue, cancellations, vehicle performance, and customer behavior using **Power BI and SQL**.

---

## Objectives

* Analyze total bookings and revenue trends
* Identify cancellation patterns and reasons
* Compare performance across vehicle types
* Evaluate payment methods and ratings

---

## Tools Used

* Power BI
* SQL
* Microsoft Excel
* DAX

---

## Dashboard Features

* Total bookings, revenue, and success rate KPIs
* Vehicle-wise booking and revenue analysis
* Payment method distribution
* Customer and driver cancellation analysis
* Driver and customer ratings analysis
* Dynamic date filters for interactive reporting

---

## DAX Usage

* Created calculated measures for total bookings, revenue, and cancellations
* Calculated success rate and cancellation percentage
* Built date-based and vehicle-based performance metrics

---

## Key Insights

* Cash and UPI are the most used payment methods
* Prime category vehicles generate higher revenue
* Most cancellations are due to operational issues
* Ratings are consistent across vehicle types
